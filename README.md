# Hista
Donkey Kong game.
# RTFM
## Required tools
- git
- make
- cmake
- C++17
- SFML
## Installation
- Clone the repository
```shell
git clone git@github.com:Cruder/hista.git
```
- Create a cmake-build-debug folder
```shell
cd hista
mkdir cmake-build-debug
```
## Build
```shell
cd bin
cmake ..
make
```
## Launch projet
```shell
./hista_project
```
